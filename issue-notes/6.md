# issue Blueskyのtimelineに対応する #6
[issues #6](https://github.com/cat2151/zenn-qiita-mute-warning/issues/6)

- URL : https://bsky.app/
- 用途
    - ZennやQiitaの投稿のtimelineにおいて、userのmute設定を反映させる用
- 仕様：timelineにて、各投稿ごとに処理を行う：
    - 投稿に、zennかqiitaのURLが含まれていたら、
        - そのURLから筆者を取得し、それがmute listにあるなら、
            - その投稿に赤いwarningのバナーを追加表示する
